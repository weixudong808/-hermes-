# 功能清单 & 迁移上下文

> 最后更新：2026-05-03

## 功能全貌

### v1.0（已测试通过，2026-05-01 验证）

| 功能 | 关键点 |
|------|--------|
| 三级消息分类 | @机器人→总结/查询/提问；训练计划→写表；日常→按风格互动 |
| 训练数据解析 | 6类30+动作，8种格式变体，模糊匹配 |
| 多维表格写入 | 串行：训练课次→record_id→动作记录（逐个） |
| 多维表格读取 | 课次表+动作表，总结生成用 |
| 总结生成 | 概况/明细/分析/建议四段式 |
| 风格适配 | energetic/professional/gentle/strict |
| 训练主题自动判断 | 动作名→主题，合并规则（上肢推/拉等） |
| 补记历史训练 | 提取日期→毫秒时间戳写入 |
| 自重动作识别 | 引体向上/平板支撑等自动标记 |
| 未映射群自动处理 | 自动创建映射+会员profile |
| 缺失字段处理 | 组数默认1+提示，待确认标记 |
| 跨群隔离 | Hermes session 级隔离，chat_id 匹配 bitable |

### v2.0（2026-05-03 讨论确认，待实施）

| 功能 | 描述 | 关键设计 |
|------|------|---------|
| 角色识别 | group_map 中 coach_user_id，教练发的才记录训练数据 | 会员发的即使有训练数据也不记录 |
| 隐私红线 | 其他会员信息/教练总会员数/剩余课时等绝不回答 | 标准拒绝话术 |
| 上下文重置提示 | 每日重置后首条消息追加"消息已重置"提示 | 仅首条，后续不重复 |
| 会员历史查询 | 触发词→查多维表格 | 不查对话上下文（不可靠） |
| 健身知识三级分区 | 绿区直接答/黄区答+引导教练/红区不答 | 总分总格式：安抚→知识→建议 |
| 饮食照片提醒 | per-member cron job，每天按时提醒 | 会员不回也发，直到课程结束 |
| 体重提醒 | 每周一次 | per-member |
| 入群问卷 | 文字形式（卡片等飞书bug修复后再美化） | 目标/体重/吃饭时间→profile.json |
| 教练控制提醒时间 | 会员不能调整 | 教练@机器人指令修改 |

## 企业微信可行性结论（2026-05-03 调研）
